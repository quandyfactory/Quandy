Quandy README

Quandy is a sweet, simple library to help you create web applications with Python. It plays nice with web.py and sqlalchemy.

  • SQLAlchemy is a fantastic ORM http://www.sqlalchemy.org/

Things I Wish Quandy Had

Unicode Support

I wrote this before I really understood how unicode works in Python (okay, I'm still not 100% sure I understand it), so it punts on unicode support. I really need to fix this.

Packaging

At some point I need to tackle packaging this up either through PyPi or using easy_install. In the meantime it's a file.

Better Forms

For sheer expedience, I wrote the form and formfield classes so that they lay forms out using HTML tables (hiss! boo!). I'd like to add an optional parameter that lets you switch between table-based layout and div-based layout.
